# BlogSimple - Project Setup

## Objetivo

Desarrollar una plataforma de blog moderna utilizando Vue 3, TypeScript y una arquitectura modular escalable.

---

# Paso 1 - Crear proyecto Vue

Crear el proyecto utilizando Vite.

```bash
npm create vite@latest blogsimple
```

Seleccionar:

```text
Framework:
Vue

Variant:
TypeScript
```

Ingresar al proyecto:

```bash
cd blogsimple
```

Instalar dependencias:

```bash
npm install
```

Ejecutar proyecto:

```bash
npm run dev
```

---
