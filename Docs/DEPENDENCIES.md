# Dependencias del Proyecto

## Vue Router

```bash
npm install vue-router
```

Uso:

* Navegación
* Rutas protegidas
* Lazy Loading

---

## Pinia

```bash
npm install pinia
```

Uso:

* Estado global

---

## Axios

```bash
npm install axios
```

Uso:

* Consumo APIs REST

---

## SCSS

```bash
npm install -D sass
```

Uso:

* Estilos escalables

---

## Jest

```bash
npm install -D jest
```

Uso:

* Testing unitario

---

## Cypress

```bash
npm install -D cypress
```

Uso:

* Testing End To End
